Conversor de Temperaturas en Python

Este proyecto es un programa interactivo en Python que permite convertir temperaturas entre Celsius, Fahrenheit y Kelvin mediante un menú sencillo en consola. El código está pensado para principiantes y estudiantes que quieran aprender sobre estructuras de control, funciones y validación de datos.

Características principales

✔ Conversión de Celsius → Fahrenheit
✔ Conversión de Fahrenheit → Celsius
✔ Conversión de Celsius → Kelvin
✔ Conversión de Kelvin → Celsius
✔ Conversión de Fahrenheit → Kelvin
✔ Conversión de Kelvin → Fahrenheit
✔ Menú interactivo fácil de usar
✔ Validación de entradas
✔ Código limpio y comentado

Tecnologías utilizadas

Python 3.x

Consola / Terminal

(Opcional) Visual Studio Code como editor de código

Ejemplo de uso
===== CONVERSOR DE TEMPERATURAS =====
1. Celsius a Fahrenheit
2. Fahrenheit a Celsius
3. Celsius a Kelvin
4. Kelvin a Celsius
5. Fahrenheit a Kelvin
6. Kelvin a Fahrenheit
7. Salir
Seleccione una opción: 1
Ingrese los grados Celsius: 25
Resultado: 25°C = 77°F

Este programa tiene como finalidad servir como práctica para:

Uso de funciones en Python
